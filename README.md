# Overview
This resource pack adds blue hearts and some visibility/PvP improvements.
# Features
Blue Hearts,
Improved visibility on poison hearts and
Improved visibility on wither hearts.
# What about hardcore?
Don't worry, because same features apply to hardcore hearts!
# Why use it for PvP?
Especially in DeBuff PvP where you have many poison potions this resource pack becomes very handy!
